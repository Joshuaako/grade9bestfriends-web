# Grade 9 Best Friends

The world's first social network dedicated to preserving school friendships from Grade 9.

## Features
- User accounts with profile photos
- Friendship groups with invite links
- Memory posts with photo uploads
- Comments on posts
- Digital yearbook for leaving messages

## Tech Stack
- Next.js 14
- TypeScript
- Supabase (Auth, Database, Storage)
- Tailwind CSS
